# 프로젝트 개요
| 항목            | 내용                                                                 |
|-----------------|----------------------------------------------------------------------|
| **프로젝트**    | WinAPI 를 활용한 게임                                               |
| **장르**        | 1인칭 요리 시뮬레이션                                               |
| **진행 방식**   | 포인트 앤 클릭, 미니게임                                            |
| **게임 설명**   | 사연을 가진 귀신 캐릭터들의 요구에 맞게 그들이 먹고자 하는 우동을 요리하여 성불시키는 게임 |
